# Personal Portfolio Website

A high-performance, responsive, and visually striking single-page developer portfolio website. Built from scratch with semantic HTML5, modern CSS3 (featuring dynamic radial gradients and CSS custom properties), and standard Vanilla JavaScript. 

---

## 🎨 Visuals & Identity

* **Theme:** Deep oceanic/teal background (`#0a3a4a`) accented by a warm amber yellow (`#f0c419`) and vibrant electric blue (`#5bc0eb`).
* **Typography:** A premium typography stack utilizing *Playfair Display* for striking, editorial headings alongside *Inter* for clean, legible body text, and *JetBrains Mono* for technical metrics.
* **Aesthetics:** Modern frosted-glass (`backdrop-filter`) navigation menus, smooth scaling cards, micro-interactions, and continuous floating animations for code stack badges.

---

## 🚀 Key Features

* **Responsive Layout:** Uses a Fluid CSS Grid and Flexbox container layout that shifts seamlessly from 4K displays down to ultra-small mobile screens.
* **Interactive Menu:** A mobile-optimized responsive navigation menu featuring an intuitive hamburger transition toggle.
* **Animated Progress Skill Bars:** Dynamic presentation data bars linked to internal layout percentages that reflect stack proficiencies.
* **Performance Optimization:** Zero external framework dependencies (No Tailwind, No Bootstrap, No React), ensuring instantaneous page loads and structural rendering.
* **Accessibility First:** Utilizes semantic structural elements (`<nav>`, `<header>`, `<section>`, `<article>`) coupled with descriptive `aria-label` tags for screen-reader support.
